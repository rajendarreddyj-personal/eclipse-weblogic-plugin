Welcome to the Eclipse Weblogic Plugin update site.

This web page provides automatic distribution and updates for the Eclipse Weblogic Plugin.

Plugin requirements

This plugin has primarily been tested with Eclipse 4.5 and 4.6 It should work with 4.x releases, but let us know if you have any problems. The plugin is not compatible with versions of Eclipse preceding 4.5. The plugin runs under Java 1.7/7.0, or newer.

Plugin installation

To install the Eclipse Weblogic Plugin:

  1. In Eclipse, click on Help -> Install New Software...

  2. Click on Add..

  3. Enter the following:

    • Name: Eclipse Weblogic Plugin Update Site
    • URL: https://rajendarreddyj.github.io/eclipse-weblogic-plugin/update-site/

    and click OK.

  4. You should see Eclipse Weblogic Plugin. Select the checkbox next to it and click Next >.

  5. You'll need to accept the license and confirm you want to install a plugin that is not digitally signed. Go ahead and install it anyway.

  6. Restart eclipse.
